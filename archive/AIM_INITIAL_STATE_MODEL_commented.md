# AIM_INITIAL_STATE_MODEL

## Formal Ontological State Model (Advaita Inquiry Matrix)

This document formalizes the default human condition in terms suitable
for system architecture, state modeling, and transition logic within
AIM.

------------------------------------------------------------------------

# I. Primary Ontological States

## STATE S0 --- Avidyā (Default Condition)

**Definition:**
Misidentification of ātman with body-mind (adhyāsa).

**Structural Properties:** 

- Dehābhimāna (body-identification) 
- Kartṛtva-bhoktṛtva (doership-enjoyership) 
- Nāma-rūpa realism (reification of appearance) 
- Mṛtyu-bhaya (fear of death) 
- Externalized validation dependency

**Behavioral Outputs:** -

- Seeking security in objects 
- Role-based identity construction 
- Hierarchical conformity 
- Fragmented cognition (thought ≠ feeling ≠ action)

**Ontological Status:**
Beginningless ignorance (anādi avidyā), experientially real
(vyāvahārika-satya).

------------------------------------------------------------------------

## STATE S1 --- Saṃsāric Stabilization

**Definition:**
Operational normalization of ignorance.

Ignorance becomes reinforced through:

-   Social mirroring
-   Collective adhyāsa
-   Institutional structures
-   Idle cognitive loops

**System Effects:** 

- Identity hardening 
- Anxiety management strategies 
- Narrative self-maintenance 
- Suppression of ontologicalquestioning

Transition to S1 from S0 is automatic.

------------------------------------------------------------------------

## STATE S2 --- Ontological Disturbance

**Trigger Conditions:** 

- Existential anxiety 
- Encounter with death (literal or symbolic) 
- Failure of external identity structures 
- Exposure to śāstra or guru-vākya

**Characteristics:** 

- Recognition of insufficiency of externals 
- Cognitive dissonance 
- Increased inquiry potential

This state is unstable and can regress to S1.

<!--“Increased inquiry potential” should be amplified and detail added; the system should initiate inquiry if the student is slow to take it up-->

------------------------------------------------------------------------

## STATE S3 --- Vicāra Initiation

**Definition:**
Deliberate engagement in Self-inquiry.

**Inputs Required:** 

- Śraddhā (trust in means of knowledge) 
- Adhikāra (preparedness) 
- Śāstra-pramāṇa 
- Structured guidance

**Primary Operation:** Viveka (discrimination between satya and mithyā).

------------------------------------------------------------------------

## STATE S4 --- Adhyāsa Deconstruction

**Process:** Systematic negation (neti-neti).

**Functional Outcomes:** 

- De-identification from body 
- De-identification from mind 
- De-identification from roles 
- Witness-recognition (sākṣitva clarity)

Ignorance weakens but may still recur.

<!--in addition to the negative outcomes (De-identification) there should be positive markers, like “Recognition of scriptural authority,” etc.-->

------------------------------------------------------------------------

## STATE S5 --- Ātma-Svarūpa-Pratyabhijñā

**Definition:**
Recognition of Self as non-dual awareness (sat-cit-ānanda).

**Structural Markers:** 

- Akartṛtva recognition 
- Asaṅgatva recognition 
- Abhaya (freedom from death-fear) 
- Ontological wholeness

No acquisition occurs --- only ignorance removal.

<!--yes, S5 should distinguish jñāna-utpatti from jñāna-niṣṭhā-->

------------------------------------------------------------------------

# II. Transition Logic

S0 → S1 : Automatic conditioning reinforcement
S1 → S2 : Existential shock or pramāṇa exposure
S2 → S3 : Acceptance of inquiry discipline
S3 → S4 : Sustained discrimination process
S4 → S5 : Knowledge assimilation (aparokṣa-jñāna)

Regression possible from any state prior to S5.

<!--make regression paths explicit so the system can recognize them-->

------------------------------------------------------------------------

# III. System Bias Analysis

Default System Bias: Toward S1 (stable ignorance).

Why: 

- Social reinforcement 
- Psychological inertia 
- Fear of dissolution 
- Reward structures based on nāma-rūpa success

Authenticity is therefore not baseline but emergent.

------------------------------------------------------------------------

# IV. Design Implications for AIM

1.  System must detect S0/S1 signatures.
2.  System must safely induce S2 (ontological disturbance).
3.  System must support structured transition into S3.
4.  System must provide precise negation logic in S4.
5.  System must stabilize recognition in S5 without ego-reconstitution.

------------------------------------------------------------------------

# V. Initial Conditions Summary (Formal)

Initial Human State = S0 (Avidyā)

Characteristics: 

- Structural misidentification 
- Anxiety rooted in temporality 
- Socially reinforced dualism 
- Functional fragmentation

Goal of System: Not behavioral modification. Not belief replacement. Not
identity enhancement.

Goal = Ignorance removal (avidyā-nivṛtti) through knowledge.
