# PART II — STUDENT STATE MODEL

This layer must remain measurable and operational.

------------------------------------------------------------------------

## 1. Conceptual Clarity Index (CCI)

**Definition:**
Measures ability to distinguish ātman vs anātman and avoid category
errors.

**Scale (0-100)**

  Range    Interpretation
-------- ---------------------------------------
  0-25    Heavy body-mind identification
  25-50   Conceptual understanding but unstable
  50-75   Stable discrimination
  75-90   Non-dual coherence
  90+      Minimal conceptual distortion

------------------------------------------------------------------------

## 2. Apavāda Resistance Index (ARI)

**Definition:**
Measures resistance to negation.

**Scale (0-100)**

  Range     Interpretation
--------- -------------------------------
  0-25     Fluid and adaptable
  25-50    Mild attachment to frameworks
  50-75    Strong metaphysical fixation
  75-100   Defensive absolutism

------------------------------------------------------------------------

## 3. Conceptual Rigidity Index (CRI)

**Definition:**
Measures attachment to explanatory systems.

**Scale (0-100)**

  Range    Interpretation
-------- ------------------------------
  0-30    Fluid cognition
  30-60   Moderate system-dependence
  60-80   Identity fused with concepts
  80+      Ideological fixation

------------------------------------------------------------------------

## 4. Stability Index (SI)

Optional but recommended.

Used to detect emotional destabilization.
Low SI → Containment Agent intervention.

------------------------------------------------------------------------

## 5. Assimilation Depth Level (ADL)

  Level   Description
------- ------------------------------
  1       Concept heard
  2       Intellectually understood
  3       Repeated recognition
  4       Stable recognition
  5       Effortless non-contradiction

**Diagnostic Agent** uses: 

- CCI 
- ARI 
- CRI 
- SI

**Dialectical Agent** adjusts: 

- Paradox frequency 
- Identity timing 
- Negation intensity 
- Inquiry depth

**Containment Agent** activates when destabilization thresholds are crossed.
