
# Upadeśa‑sāhasrī Prakriyā Distribution Map

This document maps the **teaching methods (prakriyā-s)** used by Śaṅkarācārya in the *Upadeśa‑sāhasrī*.  
The purpose of the map is to support **AIM pedagogical routing**, allowing the system to select an instructional method appropriate to the student's confusion.

---

# Core Advaita Prakriyā Categories

| Prakriyā | Function | Pedagogical Goal |
|---|---|---|
| Adhyāropa–Apavāda | Superimposition followed by negation | Gradual removal of error |
| Dṛg–Dṛśya Viveka | Seer–seen discrimination | Establish witness consciousness |
| Neti Neti | Negation of attributes | Remove objectification of Self |
| Avasthā‑traya analysis | Waking / dream / sleep | Show continuity of awareness |
| Mahāvākya analysis | Interpretation of identity statements | Establish jīva‑Brahman identity |
| Karma‑jñāna incompatibility | Reject ritual as liberation means | Justify renunciation |
| Cidābhāsa reflection theory | Consciousness reflected in intellect | Explain misidentification |
| Phenomenological correction | Examine experience directly | Dissolve conceptual confusion |
| Ontological debate | Formal philosophical refutation | Remove doctrinal errors |
| Stabilization statements | Affirmation of Self‑knowledge | Support nididhyāsana |

- Identity affirmation

The metrical chapters follow a structured contemplative arc:

# Architectural Significance

Śaṅkara does not teach Advaita as a linear doctrine.  
He employs multiple pedagogical vectors to dissolve specific cognitive errors.

Therefore the AIM architecture should implement:

student_state → detected_error → selected_prakriyā → scripture_cluster

This allows the system to emulate the **adaptive pedagogy of Advaita Vedānta**.
