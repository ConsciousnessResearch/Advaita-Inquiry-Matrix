# Minimal Advaita Curriculum

This document identifies the **smallest set of texts required to
transmit the complete pedagogical arc of Advaita Vedānta**.

The curriculum is derived from the Universal Advaita Teaching
Progression and the Advaita Corpus Teaching Map. It demonstrates that
only a small number of carefully selected texts are necessary to guide a
student through the entire inquiry process from existential questioning
to stabilized recognition.

------------------------------------------------------------------------

# Example Curriculum Flow

A classical Advaita curriculum could follow this progression:

Kaṭha 1.1\
→ existential shock (death confrontation)

Kaṭha 1.2\
→ ethical discrimination (śreyas vs preyas)

Kaṭha 1.3\
→ instrument analysis (body--mind as chariot)

Chāndogya 6\
→ ontological inquiry and **tat tvam asi**

Taittirīya\
→ progressive negation of identity layers (pañca-kośa)

Māṇḍūkya\
→ recognition of witnessing consciousness

Bṛhadāraṇyaka\
→ apophatic clarification (**neti neti**)

Upadeśa-sāhasrī\
→ stabilization of knowledge

# Key Insight

The Advaita corpus functions less like a collection of independent
philosophical texts and more like a **coordinated teaching system**.

Each text contributes a distinct pedagogical function within the larger
teaching arc.

The minimal curriculum above shows that the entire Advaita inquiry
process can be transmitted using a relatively small set of core
scriptures.
