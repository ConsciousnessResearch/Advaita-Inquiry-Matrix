# AIM Cross‑Upaniṣad Pedagogical Integration Map (Provisional)

## Purpose

This document maps the pedagogical methods (*prakriyā*) currently
represented in the AIM corpus.
It is incremental and expands as additional Upaniṣads are ingested.

The goal is not to summarize doctrines, but to catalog teaching
strategies used by the Upaniṣads so that the AIM system can select an
appropriate method for guiding inquiry.

------------------------------------------------------------------------

# Pedagogical Method Map

## 1. Renunciation and Non‑Ownership

Primary Text: Īśa Upaniṣad

Teaching Focus: - non‑possessiveness - integration of action and
knowledge - recognition of the Self in all beings

Core Movement: world → renunciation → unity

Prakriyā: īśāvāsya renunciation method

------------------------------------------------------------------------

## 2. Existential Inquiry into the Self

Primary Text: Kaṭha Upaniṣad

Teaching Focus: - confrontation with death - discrimination between the
eternal and the transient - inquiry into the nature of the Self

Core Movement: mortality → inquiry → Self‑knowledge

Prakriyā: existential inquiry

------------------------------------------------------------------------

## 3. Consciousness‑State Analysis

Primary Text: Māṇḍūkya Upaniṣad

Teaching Focus: - waking state (jāgrat) - dream state (svapna) - deep
sleep (suṣupti) - the fourth (turīya)

Core Movement: experience → awareness → pure consciousness

Prakriyā: avasthā‑traya analysis

Mahāvākya: ayam ātmā brahma

------------------------------------------------------------------------

## 4. Higher vs Lower Knowledge

Primary Text: Muṇḍaka Upaniṣad

Teaching Focus: - distinction between aparā vidyā and parā vidyā -
transcendence of ritual knowledge - realization of Brahman through
knowledge

Core Movement: ritual → inquiry → knowledge of Brahman

Prakriyā: vidyā hierarchy

------------------------------------------------------------------------

## 5. Kośa Discrimination

Primary Text: Taittirīya Upaniṣad

Teaching Focus: - body (annamaya) - vital force (prāṇamaya) - mind
(manomaya) - intellect (vijñānamaya) - bliss (ānandamaya)

Core Movement: gross → subtle → causal → Brahman

Prakriyā: pañca‑kośa discrimination

Key Teaching: satyam jñānam anantam brahma

------------------------------------------------------------------------

## 6. Cosmological Projection

Primary Text: Aitareya Upaniṣad

Teaching Focus: - the Self as the origin of the cosmos - manifestation
of the cosmic person - emergence of embodied experience - recognition of
consciousness as ultimate reality

Core Movement: Self → cosmos → individual → consciousness

Prakriyā: sṛṣṭi‑prakriyā

Mahāvākya: prajñānam brahma

------------------------------------------------------------------------

# Structural Relationships

The current corpus already reveals complementary teaching vectors:

Cosmology --- Aitareya
Phenomenological Reduction --- Taittirīya
Consciousness Analysis --- Māṇḍūkya
Existential Inquiry --- Kaṭha
Renunciation Framework --- Īśa
Knowledge Hierarchy --- Muṇḍaka

Together these provide multiple independent pedagogical approaches
leading to the same realization.

------------------------------------------------------------------------

# Future Expansion

As additional Upaniṣads are ingested, new pedagogical branches will
appear, especially:

-   Identity teaching: Chāndogya (tat tvam asi)
-   Neti‑neti negation: Bṛhadāraṇyaka
-   Structured inquiry dialogue: Praśna

This map will expand accordingly.

------------------------------------------------------------------------

# Role in AIM

This map functions as a teaching‑strategy index.

When analyzing a student's question, AIM can select a pedagogical method
appropriate to the student's current misunderstanding.
