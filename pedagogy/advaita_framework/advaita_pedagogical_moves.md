
# Advaita Pedagogical Moves (AIM)

This document catalogs the primary teaching operations used in Advaita Vedānta.
These are not doctrines but **pedagogical moves** used by a teacher to remove
ignorance (adhyāsa) and guide the student toward recognition of the Self (ātman).

These moves appear repeatedly across the Upaniṣads, the Brahma‑sūtra,
and Śaṅkarācārya’s commentaries.

---

# 1. Adhyāsa Correction

**Purpose:** Remove superimposition of non‑Self upon the Self.

Typical formulation:

    "You are confusing the Self with the body, mind, or senses."

Pedagogical logic:

    superimposition detected
    ↓
    teacher reveals confusion
    ↓
    student withdraws identification

Typical sources:

- Brahma‑sūtra Bhāṣya introduction
- Bṛhadāraṇyaka Upaniṣad dialogues

Clusters:

```
adhyasa_correction
identity_analysis
```

---

# 2. Neti‑Neti Elimination

**Purpose:** Negate all observable phenomena as the Self.

Formula:

    neti neti
    "not this, not this"

Pedagogical logic:

    observe object
    ↓
    recognize object is known
    ↓
    knower cannot be object
    ↓
    negate identification

Primary source:

- Bṛhadāraṇyaka Upaniṣad

Clusters:

```
negation_method
identity_analysis
```

---

# 3. Kośa Discrimination

**Purpose:** Separate the Self from successive layers of embodiment.

Sequence:

```
annamaya
prāṇamaya
manomaya
vijñānamaya
ānandamaya
```

Pedagogical logic:

    layer observed
    ↓
    layer recognized as object
    ↓
    not the Self

Primary source:

- Taittirīya Upaniṣad (Ānandavallī)

Clusters:

```
kosha_analysis
identity_analysis
```

---

# 4. Witness Recognition (Sākṣī)

**Purpose:** Shift identity from mental activity to awareness itself.

Teaching move:

    thoughts appear
    ↓
    awareness knows thoughts
    ↓
    awareness is not thought

Typical formulation:

    "You are the witness of the mind."

Primary sources:

- Māṇḍūkya Upaniṣad
- many Advaita prakaraṇa texts

Clusters:

```
witness_recognition
direct_inquiry
```

---

# 5. Identity Revelation

**Purpose:** Reveal the identity of the individual Self and Brahman.

Classic formulations:

```
tat tvam asi
aham brahmāsmi
ayam ātmā brahma
prajñānam brahma
```

Pedagogical logic:

    remove false identification
    ↓
    reveal underlying identity

Primary sources:

- Chāndogya Upaniṣad
- Bṛhadāraṇyaka Upaniṣad

Clusters:

```
mahavakya_revelation
nondual_identity
```

---

# 6. Contemplative Verification

**Purpose:** Encourage the student to verify the teaching through inquiry.

Typical instruction:

    tapasā brahma vijijñāsasva

Pedagogical logic:

```
teaching heard
↓
student contemplates
↓
insight confirmed
```

Primary source:

- Taittirīya Upaniṣad (Bhṛguvallī)

Clusters:

```
contemplative_inquiry
experiential_verification
```

---

# 7. Realization Expression

**Purpose:** Allow realization to express itself spontaneously.

Examples:

```
aham annam
aham brahmāsmi
śivo'ham
```

These are **not practices**, but poetic expressions of knowledge.

Clusters:

```
realization_expression
cosmic_identity
```

---

# Summary

Advaita teaching progresses through a sequence of pedagogical moves:

```
adhyāsa detection
↓
identity discrimination
↓
negation (neti neti)
↓
witness recognition
↓
mahāvākya revelation
↓
contemplative verification
↓
realization expression
```

These moves form the operational teaching logic used throughout the Advaita corpus.
