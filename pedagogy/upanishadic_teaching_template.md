# Universal Upaniṣadic Teaching Template

## Overview

Most Upaniṣadic dialogues follow a recognizable instructional arc
designed to remove **adhyāsa** (superimposition) and reveal the identity
of **Ātman and Brahman**.

The teaching proceeds through several pedagogical phases.

diagnosis → metaphysical orientation → experiential analysis → identity
revelation → stabilization

Each stage addresses a specific cognitive obstacle in the student.

------------------------------------------------------------------------

# 1 Diagnostic Phase (Ignorance Exposure)

## Function

Expose the student's misunderstanding or intellectual pride.

Typical strategies:

-   questioning the student's knowledge
-   revealing limits of scriptural learning
-   triggering epistemic humility

## Example

Chāndogya Upaniṣad 6.1

Uddālaka observes Śvetaketu's arrogance.

"Did you ask for that knowledge by which the unheard becomes heard?"

### AIM Tag Pattern

``` yaml
pedagogical_role: diagnostic_entry
pedagogical_stage: ignorance_exposure
```

------------------------------------------------------------------------

# 2 Ontological Orientation (Metaphysical Grounding)

## Function

Introduce the fundamental metaphysical principle.

In Advaita this is usually:

kārya--kāraṇa-ananyatva

(effect is non-different from cause)

## Example

Chāndogya Upaniṣad 6.1--6.2

mṛttiketyeva satyam --- "Clay alone is real."

### AIM Tag Pattern

``` yaml
cluster: cause_effect_identity
prakriyā: kārya-kāraṇa-ananyatva
```

------------------------------------------------------------------------

# 3 Cosmological Projection (Explaining Multiplicity)

## Function

Explain how the apparent world emerges from the single reality.

This neutralizes the student's objection:

"If everything is one, why do we see many?"

## Example

Chāndogya Upaniṣad 6.3--6.4

sat → tejas → ap → anna

### AIM Tag Pattern

``` yaml
cluster: cosmological_projection
prakriyā: satkāryavāda
```

------------------------------------------------------------------------

# 4 Microcosm Analysis (Body--Mind Explanation)

## Function

Relate the metaphysics of the universe to the human organism.

This prepares the student to see the Self directly.

## Example

Chāndogya Upaniṣad 6.5--6.7

food → mind\
water → prāṇa\
fire → speech

### AIM Tag Pattern

``` yaml
cluster: embodiment_analysis
prakriyā: element_to_function_mapping
```

------------------------------------------------------------------------

# 5 Experiential Demonstration (Verification through Observation)

## Function

Shift from abstract metaphysics to observable experience.

Often includes experiments or demonstrations.

## Example

Chāndogya Upaniṣad 6.7

Śvetaketu fasts for fifteen days, demonstrating that mind depends on
food.

### AIM Tag Pattern

``` yaml
cluster: empirical_demonstration
prakriyā: experiential_verification
```

------------------------------------------------------------------------

# 6 Identity Teaching Loop (Mahāvākya Assimilation)

This is the core engine of Upaniṣadic pedagogy.

The teacher repeatedly runs the loop:

analogy → identity statement → request for clarification

## Structure

analogy\
↓\
mahāvākya\
↓\
student requests explanation\
↓\
new analogy

## Example

Chāndogya Upaniṣad 6.8--6.16 repeatedly declares:

tat tvam asi śvetaketo


``` yaml

```
